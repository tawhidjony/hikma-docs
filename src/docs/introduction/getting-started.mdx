# Getting Started with Hikma UI

Hikma UI is a React component library that integrates seamlessly with React Hook Form and Zod for form validation. This combination provides a powerful toolkit for building forms with robust validation in your React applications.

## Prerequisites

Before getting started, make sure you have the following installed:

- Node.js (version 14 or higher)
- npm or yarn

## Installation

Install Hikma UI and its peer dependencies:

```bash
npm install hikma-ui
```


## Importing Components

To use Hikma UI components in your project, you need to import them from the library. Here's an example of how to import and use a button component:

```tsx
import { UiButton } from 'hikma-ui';

<UiButton variant="primary">Click me</UiButton>
```
