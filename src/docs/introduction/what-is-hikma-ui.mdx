# What is Hikma UI?

Hikma UI is a modern, accessible React component library designed specifically for building beautiful Islamic applications. The name "Hikma" comes from the Arabic word meaning "wisdom", reflecting our goal to create thoughtful, well-designed components that serve the Muslim developer community.

## Key Features

- **Islamic-First Design**: Components are crafted with Islamic applications in mind, providing specialized UI elements for prayer times, Quranic text display, and more
- **Modern React**: Built using the latest React patterns and best practices
- **Fully Customizable**: Easily theme and style components to match your brand
- **Accessible**: Built with accessibility in mind following WAI-ARIA guidelines
- **TypeScript Support**: Full TypeScript support for enhanced developer experience
- **Responsive**: Mobile-first design approach ensures great experience across devices

## Why Hikma UI?

While there are many great component libraries available, Hikma UI fills a specific need in the Islamic software development space. Our components are built with consideration for:

- Right-to-left (RTL) text support
- Islamic calendar integration
- Prayer time calculations and displays
- Quranic text rendering
- Halal/Islamic-compliant UI patterns

Whether you're building a mosque website, Islamic education platform, or any Muslim-focused application, Hikma UI provides the building blocks you need to create beautiful, functional interfaces.
